## 3:00–4:00 — likely Q&A answers

**Why is this not just translation plus screenshots?**

> Translation preserves sentences. nativas makes component-level business decisions using visual hierarchy, market evidence, and reviewed localization precedents, then has a separate Hermes specialist challenge meaning drift and unsupported claims.

**What exactly is autonomous?**

> After URL submission, Hermes plans the job, selects and delegates specialists, gathers bounded evidence through tools, reviews their work, repairs weak findings, publishes the report, and carries the context into a paid continuation. Humans handle exceptions, not every step.

**What is defensible?**

> The compounding asset is a reviewed, context-rich KR/US golden set tied to component type, audience, industry, screenshots, and observed outcomes. New audits create candidates, but nothing enters the trusted set without evaluation, so improvement does not become an automated hallucination loop.

**What happens when a site blocks automation or lacks a locale pair?**

> The run returns an explicit exception with captured evidence and a recovery path. It never invents the missing page or silently lowers confidence.

**Why Hermes?**

> The work is not one prompt. It is a managed service engagement: inspect, plan, delegate different expertise, use tools, challenge the draft, revise, publish, and continue with memory. Hermes is the operating layer that makes that organization visible and auditable.
