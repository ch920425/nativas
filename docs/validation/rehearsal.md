# Rehearsal and judge-proof runbook

## Two-minute path

Before judging, prove one successful real Linkup `standard` query. Then submit the prevalidated homepage; show real audit/Hermes IDs and genuine capture/KB/Linkup/delegation events. While it runs, transparently open a report produced by the same system immediately before judging. Show paired screenshots and three findings with evidence/KB refs. Complete real Dodo test checkout, then show the verified event creating a new paid audit and Hermes run without human approval.

## Emergency path

Show a previously completed real run with Hermes trace, Convex events/report, Browser Run/R2 artifacts, Linkup evidence, gbrain query, Dodo event, and paid run. State explicitly that it is prior deployed-system output, not a live execution.

Before staging, run G0–G10, record IDs/timestamps, verify service URLs/test credentials and the fixture's direct locale URLs, and freeze features at T+170. After freeze, fix only P0 configuration, security, or fixture defects.
