# Deterministic acceptance gates

| Gate | Pass criteria | Failure action |
|---|---|---|
| G0 Contracts | v1 schemas/fixtures validate; no shadow types | Stop lane coding |
| G1 Hermes | `HERMES-01..03`: pinned `doctor`, authenticated Run/SSE, real 2–3-child flat delegation, crash-boundary no-duplicate proof | Repair by T+25; no Hermes-free fallback |
| G2 Capture | Safe locale pair and four artifacts/page; SSRF/redirect suite rejects attacks | Fixture allowlist; no alternate crawler |
| G3 KB | Six reviewed records reproduce; six queries hit correct type top 3; fallback works | Disable embeddings only; no fake refs |
| G4 Linkup | Real `standard` result ≤3 cited items within 12s | Mark degraded/use gbrain; never Exa |
| G5 Free run | One parent/batch, ≤1 real repair, exactly three valid findings ≤240s | Reduce children 3→2, not findings |
| G6 UI | First real event ≤5s; ordered refresh recovery; no fabricated child state | Show truthful coarse tool activity |
| G7 Payment | Verified test payment; duplicate delivery creates exactly one linked paid audit and one bound/running Hermes run | Fix verification; no manual bypass |
| G8 Security | Secrets absent; loopback/allowlist/service auth/raw signature/bounds pass | Block release |
| G9 Quality | All enumerated P0 IDs pass, risk score ≥90, and required branch coverage ≥90% | Cut features, not gates |
| G10 Rehearsal | Full free→pay→paid-start once; second free run; emergency path | Demo only verified scope |

All gates pass. Only G4 may use its documented degraded path if a prior real Linkup smoke exists. No other gate can be replaced with canned output. A completed paid report is P1 and is not required by G7 or G10.
